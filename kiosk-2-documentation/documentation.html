    1- Getting Started

    IMPORTANT: Don't use sub-folders as root directories of your website as it can cause some scripts and features of WordPress to fail. If you must install to a sub-directory for technical reasons, you must use this procedure.
    CORRECT: www.website.com -OR- www.your.website.com
    INCORRECT: www.website.com/wordpress/

    WordPress is a robust, completely free self-publishing platform. This guide will take you through basic setup and configuration of WordPress on your own web host (also called a "wordpress.org" install). Please note, if you have a wordpress.com account, you will only be able to install and use the Obox Themes available from the approved wordpress.com theme directory and should refer to the wordpress.com documentation for help and support.

    Note for Existing Installations

    If you are installing your theme to an existing WordPress installation that already has content, we strongly recommend you deactivate all plugins prior to activating the theme. Do not reactivate them until you are finished configuring the theme and widgets. This assures you will not run into conflicts or difficulties caused by an incompatible plugin while setting up your theme.

    Parts of your content may no longer appear or look the same once you have switched themes. If you were using shortcodes, styles or custom fields specifc to your previous theme, you will need to adjust your content to be compatible with the theme. Please refer to your previous theme's documentation or support for details or questions on what you might need to change.

     Install WordPresss

    1. Check out these recommendations for a Secure WordPress installation, and also go through WordPress' Getting Started guide.
    2. Install WordPress by uploading the contents of the Wordpress download folder to your website root. It should look something like this when finished:

    3. Log in to your WordPress admin panel. You will be greeted with the WordPress Welcome page, which includes a tour and some valuable links. If you are new to WordPress, please take some time to go through this information. or visit First Steps With WordPress.
    4. Go to Settings > Permalinks, select Post Name and click Save.
    5. For existing WordPress installations: Go to Settings > Reading and ensure Latest Posts is selected.

    2-5 Install WooCommerce

    This eCommerce theme uses the popular WooCommerce plugin to provide shop functionality and inventory management. The eCommerce theme will not function without it.

    Automatic installation

    Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t even need to leave your web browser.

    1. 1. Login to your WordPress admin panel, navigate to the Plugins menu and click Add New.
    2. 2. In the search field type “WooCommerce” and click Search Plugins.
    3. 3. Once you’ve found the plugin you can view details about it such as the the point release, rating and description. Most importantly of course, you can install it by simply clicking Install Now. After clicking that link you will be asked if you’re sure you want to install the plugin. Click yes and WordPress will automatically complete the installation.

    It’s possible at this stage that you will be taken to a page requesting the FTP details of your webserver. This will be the case if you’ve not installed a plugin on this particular installation of WordPress before. If so, simply fill in your details to complete the installation.

    Manual installation

    The manual installation involves downloading the plugin and uploading it to your webserver via your favourite FTP application.

    1. Download the plugin to your computer and unzip it
    2. Using an FTP program, or your hosting control panel, upload the unzipped plugin folder to your WordPress installation’s wp-content/plugins/ directory.
    3. Activate the plugin from the Plugins menu within the WordPress admin.

    Important Info Visit the WooCommerce Docs website for more information.

    4 Setup Post Categories

    Before we can really get started with your theme, you need add some content. Categories are used to create a logical structure for your content and are used as filters in many widgets and plugins.

    Setup Categories

    1. Click on Posts → Categories.
    2. Add a new category to serve as your main Blog or News page by typing the category name, then click Add New Category.
    3. Repeat to add a category named Featured. You can use the Featured category to hand-pick posts from across multiple categories to display in your Home page slider or other widgets.
    4. Add any additional Blog or News categories to sort your posts into logical topics. This is not required, but is helpful for Search Engine Optimization. Categorizing posts also allows you to display separate blog feed pages for each category, if desired. View our Theme Demo for an example of how categories can be used.

    Useful information

    Learn more about Categories here.

    View the How To section for details on creating standard Posts, such as Adding Featured Images or Videos.

    5 Setup your Menu(s)

    Your theme may attempt to display your navigation menu automatically, but it may show items you don't wish to appear, or put them in an order you don't like. You may customize your menu using the following steps:

    1. Go to Appearance → Menus.
    2. Click the plus (+) tab to add a new menu, give it a name and click Create Menu.

      3. If you have an existing WordPress install, remove any links that are no longer relevant, such as a Blog page from an old page template.

    3. Select your new menu from the Primary Navigation drop-down menu on the left side.
    4. Create a Home link by typing your URL into the Custom Link field on the left side and adding "Home" to the label field (or whatever you wish to call it.). Click Add to Menu to add it to the menu panel.

    5. Check the pages you wish to add under the Pages box on the left and click Add to Menu. If you have a lot of Pages, you may need to click the View All link.
    6. Scroll to the bottom and check your blog category in the Category box to create your blog page. This will show all posts in chronological order using the full blog/category layout. You may also choose to add individual categories instead, and use your Archives page exclusively for showing all posts.
    7. Arrange your menu by dragging and dropping the link bars in your menu panel. Learn more about the Appearance Menus page here.
    8. Repeat these steps to add a menu for your foote. if required by the theme (you will see a Secondaryr Navigation menu on the upper left side)r. Typically this contains an alternative set of links, or a simplified menu of your top-level links only (do not add sub-pages). Once created, select this menu from the Secondary Navigation menu drop-down and click Save.

      That's it! View your page to see your results and fine-tune as needed.

    Kiosk is currently designed to promote a single product, such as an ebook, charity, event, new release, etc.  Therefore, it will display the last featured product added on the home page.. Kiosk is not a full shop theme at this time.
    1. Click on Products > Add New.
    2. Enter a Product title and short description.
    3. Configure your product options below the post editor, such as price and so on.

    5. Click Set Featured Image to upload your featured image if you are using WordPress Featured Images, or click Browse in the theme options panel to upload an Obox Main Image. You may optionally add a video embed code to use a video as the main image/presentation.

    Kiosk: Configure General Options

    Start by clicking  on Theme Options in the left-hand admin menu and choose General

    All Obox Themes come with some General options for setting a custom logo and favicon, and other customization options like colors, meta tags and so on. Here we will explain each option:

    Custom Logo

    This option allows you to upload a Custom Logo to replace the default theme logo or banner. The dimensions are purely up to you, however it should not exceed 320 pixels wide, or 300 pixels tall (for best compatibility with mobile devices). Click on Browse to upload an image file from your hard-drive, or enter the URL of your image file if it is hosted elsewhere.

    You can upload as many logos as you like and then simply switch to the one you like best by clicking on the thumbnail. WordPress accepts .jpg, .png and .gif.

    Custom Favicon

    This option allows you to upload a favicon, which is the tiny icon displayed in the browser address bar when visitors view your site. The dimensions should be 16 x 16 pixels. Click on Browse to upload an image file from your hard-drive, or enter the URL of your image file if it is hosted elsewhere.

    You can upload as many favicons as you like and then simply switch to the one you like best by clicking on the thumbnail. For best results, your favicon should be a .gif. You should not use a .jpg as it will probably not display.

    Product Featured Image

    This option determines how your product will appear on the main page. Slider displays your featured image, followed by any additional images uploaded to the product gallery via the post editor using the default WordPress image uploader. The thumbnail option will display all images in a grid.

    Thumbnail Options

    The Thumbnail Settings menu allows you to designate which option you want to use to manage the dynamic image features of the theme for posts, products and pages. 

    The Obox Main Image option uses our custom OCMX featured image function to generate beautifully resized images for display in all dynamic image areas such as widgets, sliders, portfolios and post lists. This option also allows you to take advantage of the OCMX image adjustment and effects. If this is a new installation or you are new to WordPress, keep this option selected for best results.

    The WordPress Thumbnail Feature tells the theme to use the inherent WordPress image resizing capabilities via the Set Featured Image option in your post editor. This option is best if you have an existing WordPress installation and have existing content using the WordPress featured images.

    The following figure shows the relationship between the option you choose and how the featured image must be uploaded in the post editor:

    For those of you who are switching to Kiosk from a theme that had a special custom image field, the Custom Image Meta option allows you to enter the field name into the Custom Meta field to attempt to pull in your existing featured images. The full image path must be set in your field for this option to work, ie http://www.yoursite.com/wp-content/uploads/etc/image.jpg.

    Custom RSS URL

    Enter the url to a custom RSS feed URL such as Feedburner. If you are not using a special RSS manager, you can leave this blank

    Custom Footer Text & Hide Obox Logo

    The Custom Footer Text and Hide Obox Logo section give you some control over the copyright area of the footer. All of our themes come with an Obox Logo at the bottom right of the footer. To switch this off, simply uncheck the box (although you get extra awesome points for leaving it on!) Replace our default copyright text with your own. For example: "Copyright © YourSiteName". Don't put code in this box, it won't work!

    Site Analytics

    If you would like a quick and painless way to implement Google Analytics, paste your preferred tracking code into this text area. Your code will then be insterted into the footer of the theme, which is best for performance reasons. If you would like your code in header, then you will need to insert it into the Theme Header under Appearence → Editor. Learn more about Google Analytics here.

    Kiosk: Install

    IMPORTANT: If you downloaded from Themeforest: Unzip the file downloaded from Themeforest before uploading it anywhere. Inside are some extras and the theme package, called kiosk_single.zip.

    Installing via FTP

    1. Unzip the kiosk_single.zip (or handmade-ecommerce_child.zip) to find the   theme folder.
    2. Connect to your web space using an FTP client and upload the theme folder to the wp-content/themes/ directory.
    3. Log in to your WordPress admin panel and go to Appearance > Themes
    4. Locate the theme thumbnail and click the Activate link.

    Installing via WordPress

    Note: If you are not running the latest version of WordPress, some menu options and screenshots may vary.

    1. Log in to your WordPress admin panel and go to Appearance > Themes
    2. Click the Install Themes tab at the top.
    3. Click the Upload link

    4. Click the Browse button and navigate to the location of your handmade_single.zip file. Select the file and click Open.
    5. Click Install Now to begin the installation.
    6. Wait for the success message, then click Activate.

    Set your folder permissions

    1. Connect to your web server via FTP or with your web host control panel.
    2. Browse to wp-content > themes > kiosk and open the functions folder.
    3. Right-click the cache folder and choose the CHMOD or Set Permissions option (the option name will vary depending on your software)
    4. Set the permissions to 777 (full rights).
    Example:

    If you are installing your theme via an FTP Client, please make sure that the path is wp-content/themes/kiosk/ and not wp-content/themes/kiosk_single/kiosk (not inside a subfolder of the themes directory) as this will affect the ajax and commenting of your theme.

    Add a Featured Video

    All Obox themes have the ability to display videos from YouTube, Vimeo or any source that supports embeds, which will be resized specifically for the templates and widgets.Videos can be added using the oEmbed link, or the video embed code field. Using the embed code field offers more control in terms of how the video looks when pulled in.

If hosting your own videos, you may use the WordPress media manager to upload Flash videos. Other video formats require a custom video player and modification to the theme, which is outside the scope of this documentation.

    For Tumblog themes, you must paste the embed code into the Tumblog Custom Settings panel, not the Waves Options.

    Using oEmbed (URL)

    If your video service supports the oEmbed code, then paste the URL of your video into the Video Link field. Here is a list of sites that are supported by the oEmbed function.

You must upload a video in each format to both self-hosted video fields in the theme options panel on the post-editor. For Flash fallback, paste your flash embed code into the embed field.     If you want to use an image which is in your WordPress media library then take the following steps:

    How to Update Your Theme

    All OCMX 2.0+ themes come with an auto-update feature which allows you to download and install updates with the click of a button. If you have modified your theme, you may still review each update and install them manually by checking your theme's Hotfix Page, linked from the theme's forum sidebar.

    Special Note for eCommerce Themes:

    Always update your base theme first!  The most fool-proof way is to temporarily activate your base theme, then follow the below instructions to update. When complete, re-activate your eCommerce theme and repeat these steps using Theme Options » Update Child Theme.

    To update your theme using the updater:

    1. Backup your custom.css and any other modified theme files.
    2. Go to Theme Options » Update and check your installed theme version in the right corner.
    3. Click Install next to each update you need to get up-to-date.
      4. For example:

       if your version is 1.2.0 and there is 1.2.1 and 1.2.2 available, click install for 1.2.1, then proceed with 1.2.2 and so on.

    4. Wait for the update to finish before proceeding to other updates.
    If you had previously modified theme files that were overwritten by an update, you must move the changes into the new file. Do not overwrite or restore updated theme files with older versions, or the theme may break!

    Update Manually:

    1. Go to Theme Options > Update to check your current theme version in the right corner.
    2. Visit your theme's Hotfix page (linked from the theme forum) and download each hotfix you need.
    3. Unzip each file into a temporary folder.
    4. Connect to your website via FTP and upload the files inside the temp folder to your theme directory (such as wp-content/themes/gigawatt)

    If you have issues updating, or if an update fails, you can safely reinstall your theme by downloading a fresh copy, unzipping it, and uploading the theme folder to wp-content/themes. Please backup your custom.css prior to reinstalling to retain any custom styles. You may safely restore the custom.css after reinstalling.

    You may notice your featured images on the main page, product pages and blog post appear stretched or in poor quality,even if you have uploaded large image sizes.     1. Verify you are uploading images that are larger than 580px wide.
    2. Visit your WooCommerce Settings and set your thumbnail sizes. Visit How to Setup WooCommerce under How To.

    Where are the Shortcodes?

    Shortcodes are little tags you may enter into your posts and pages which help build specially-styled content or pull in extended features or elements. The problem with building shortcodes into a theme is if you ever change themes, all of your content breaks as the shortcodes are no longer active. Rather than leave you in this situation, we don't build shortcodes into the theme. There are so many great plugin authors out there that already provide some excellent shortcode functionality. To view available plugins, go to Plugins > Add New and search for "shortcodes." You may also check out the Shortcode Ultimate plugin, which provides just about everything you could ask for.
